# Sentinel — Canal de Noticias AI Automatizado

Sistema que lee RSS feeds, filtra por keywords, analiza con Gemini y publica videos en YouTube automáticamente.

**Status:** Active — Fase 11 en progreso
**Key dates:** Daily Digest se ejecuta a las 05:00 UTC todos los días

## Objetivo actual (Fase 11)
- Videos reales de noticias como B-Roll
- Lower thirds estilo noticiero
- Guion de 10+ minutos para el Digest

## Pendiente
- Debug miniaturas YouTube (se generan pero no llegan a YouTube)
- Scripts de deploy para lower thirds y videos reales
